Niyə Code Academy-də Sistem Administratorluğu təhsili?

Bu tədris proqramı sektorun ehtiyaclarını nəzərə alaraq, Microsoft-un tədris proqramı əsasında hazırlanıb. Burada məqsəd yalnız mütəxəssis hazırlamaq deyil, müəssisələrin sistem və şəbəkə təminatına olan ehtiyaclarını tam nəzərə alıb, məsələlərin həllini qısa zamanda tətbiq edən peşəkarlar yetişdirməkdir. Code Academy-nin ən önəmli cəhəti məzunlarının texnologiya sahəsində peşəkar kimi qəbul olunmalarını və karyeralarını təkmilləşdirmələrini təmin etməsidir. Beləliklə, onlar sistem, server, bulud

(cloud), virtuallaşdırma, şəbəkə və təhlükəsizlik infrastrukturu sahəsində fəaliyyət göstərə bilərlər. Code Academy-də verilən təhsilin 85%-i praktika üzərində qurulub. Burada məqsəd sektorda hər gün artmaqda olan peşəkar qıtlığının qarşısını almaqdır. Tələbələr proqramın sonunda əldə etdikləri biliklə MTA, MCSA, MCSE sertifikatlarına sahib ola bilərlər. Bunun sayəsində onlar Azərbaycan və dünyada ehtiyac duyulan sistem və şəbəkə administratoru olaraq ixtisaslaşa bilərlər.

Kim iştirak edə bilər?

Sistem administratoru olmaq üçün riyaziyyat və ya mühəndislik ixtisası üzrə ali təhsilli olmaq vacib deyil. Tədris proqramına qatılan tələbələrin gələcəkdə sistem və şəbəkə təminatı üzrə mütəxəssis olmaq üçün istəkli olmaları kifayətdir. Proqrama Microsoft-un ən son texnologiyalarından istifadə edərək, texniki infrastrukturu tam olaraq dizayn etmək, qurmaq, idarə etmək, informasiya texnologiyaları həllərinə və bu sahədə ən son tredlərə bələd olmaq istəyənlər qoşula bilər.

Təhsil proqramının məqsədi nədir?

Təhsil proqramının məqsədi Sistem administratorluğu sahəsində ehtiyac duyulan peşəkar yetişdirməkdir. Laboratoriya məşğələlərinə əsaslanan təhsil modeli ilə ən yeni texnologiyaları öyrədən, eyni zamanda MCSA, MCSE sertifikat imtahanlarına hazırlaşan Code Academy tələbələrə beynəlxalq səviyyəli peşəkar mütəxəssis adı qazandırır.
