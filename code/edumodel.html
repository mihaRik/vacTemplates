Təhsil Modeli

Code Academy-nin təhsil modelində tələbələr nəzəri bilikllər əldə etməklə yanaşı, təcrübə keçmək imkanı da əldə edirlər. Yalnız Code Academy-də tətbiq olunan bu üsul dərslərin daha effekitv qavranmasına yönəlib. Real layihələr üzərində aparılan praktiki işlər isə tələbələrin karyeralarına tam hazır olmalarını təmin edir. Təhsilini bitirən tələbələr müəyyən müddətdə mentorların rəhbərliyi altında buraxılış layihəsi üzərində işləyirlər. Layihələr tələbələrlə real praktika qazanmağa, şəxsi portfolio-larını tərtib etməyə və uğurlu karyera qurmağa zəmin yaradır.

Təhsil modelinin əsas xüsusiyyətləri:

- Code Academy-yə məxsus Zəmanətli Təhsil Modeli;

- Mövzuların praktiki tətbiqinin dərhal həyata keçirilməsi;

- Mütəmadi olaraq praktiki seminarların təşkil edilməsi;

- Hər tələbənin tədris zamanı kompüterlə təmin olunması;

- Təhsil proqramından sonra 1-2 ay müddətində mentor nəzarəti altında buraxılış layihəsinin hazırlanması;

- Keçilən dərslərin daha yaxşı qavranması üçün şirkətlərdə real təcrübələrin təşkili;

- İngilis dili səviyyəsinin yüksəldilməsi üçün əlavə dərslərin keçilməsi;

- CV-lərin tərtib edilməsi, hazırlanan CV-lərin şirkətlərə göndərilməsi və görüşlərin təşkili.
