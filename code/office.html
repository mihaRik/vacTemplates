Niyə Code Academy-də Microsoft Office təhsili?

Code Academy-də keçilən MS Office tədris proqramı sektorun ehtiyaclarını nəzərə alaraq Microsoft-un tədris proqramı əsasında hazırlanmışdır. Təhsilin 85%-i praktiki iş üzərində qurulmuşdur. Məqsəd sadəcə MS Office-in incəliklərini öyrətmək deyil, eyni zamanda müəssisələrin MS Office ehtiyaclarını professional şəkildə qarşılayacaq

mütəxəssislər yetişdirməkdir. Code Academy-nin Zəmanətli Təhsil Modeli ilə hər bir tələbəyə təhsilini təkrarlama imkanı verilir. Mövzuların praktiki tətbiqinin dərhal həyata keçirilməsi üçün hər bir tələbə sinifdaxili kompüterlə təmin olunur.

Kim iştirak edə bilər?

Microsoft Office tədris proqramı üçün yaş limiti yoxdur. İş həyatının hər sahəsində istifadə olunan tətbiqləri daha yaxşı mənimsəyən və ya kompüterdən istifadə etməyi bilən, özünü təkmilləşdirmək istəyən hər bir kəs bu proqramda iştirak edə bilər.

Təhsil proqramının məqsədi nədir?

İş prosesində əvəzi olmayan kompüterdən daha məhsuldar istifadə üçün işçilərin karyeralarında inkişfa yardım etmək təhsil proqramının başlıca məqsədidir. Hesabat çıxartma, təqdimat hazırlama, effektiv e-mail yaza bilmə və vaxtdan məhsuldar istifadə etmə qazanacağınız qabiliyyətlərdən bəziləridir.
